VS Code Live Server Extension Setup

to Install and Use VS Code's Live Server Extension

Install | Use | video

Installation instruction

  1. In Visual Studio Code, click the Extensions icon in the left toolbar.
  2. In the search that appears, type Live Server.
  3. Select the Live Server extension. It should be the top choice.
  4. In the page for Live Server that , click the green Install button.

Usage Instructions

  • Studio Code must be open to a folder to use Live Server.
  • In VS Code's Explorer, right click (Windows) or Control+click (macOS) in an HTML file. From the context menu that appears, select Open with Live Server.
